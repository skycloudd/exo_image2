How to use

  1. Upload an image
  2. Click "Generate"
  3. Wait for the image to be processed
  4. Click "Download"
  5. Use the "Import level" option in Exoracer to import the .exolvl file
  6. IMPORTANT: Copy the newly created pattern into your own level!
  1. Upload a GIF
  2. Click "Generate"
  3. Wait for the GIF to be processed
  4. Click "Download"
  5. Use the "Import level" option in Exoracer to import the .exolvl file
  6. IMPORTANT: Copy the newly created pattern into your own level!
